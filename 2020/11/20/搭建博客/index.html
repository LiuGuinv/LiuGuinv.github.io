一. github 创建新仓库

1. 创建新仓库

新仓库名必须为 你的用户名.github.io，勾选创建 README 文件，之后个人博客网址为https://你的用户名.github.io/。

2. 进入仓库的 Setting 设置


可看到自己的博客网址。

二. 安装 HEXO

Hexo 支持 GitHub Flavored Markdown 的所有功能，快速、简洁且高效的博客框架。
